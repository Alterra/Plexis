Events

Plexis comes equipped with an Event Handler, which allows callbacks to made once a particular action is made. This allows for easy extending of your application without your clients having to modify any source files.

Loading the Events Handler

To Initialize the Events class, run this snippet:

$this->Events = load_class('Events');

Registering a Callback

To register a function or class method to recieve a callback once the event has been triggered:

$this->Events->register('$event_name', '$callback');
Arguments:

  1. $event_name [Type: String]: The name of the event that we are registering to recieve a callback on.
  2. $callback [Type: String | Array]: The function name, or an array of (Object, Method Name) to be called when the event is fired.

Triggering an Event

To trigger an event:

$this->Events->trigger('$event_name', '$args');
Arguments:

  1. $event_name [Type: String]: The name of the event that we are triggering.
  2. $args [Type: Array]: An array of variables that will be passed as parameters to the callback function / method.
