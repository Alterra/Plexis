    This method sets a redirect header, and status code.

    Redirect(string $location, int $wait = 0, int $status = 301) : void
    Static

    When this method is called, if the $wait param is greater then 1, headers will be sent.

    Parameters

    $location

    string

    The redirect URL. If a relative path is passed here, the site's URL will be appended

    $wait   [Default = 0]

    int

    The wait time (in seconds) before the redirect takes affect. If set to a non 0 value, the page will still be rendered. Default is 0 seconds.

    $status   [Default = 301]

    int

    The redirect status. 301 is moved permanently, and 307 is a temporary redirect. Default is 301.
