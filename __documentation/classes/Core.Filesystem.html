Returns

  • 'name' => "Name of the file / folder"
  • 'path' => "/path/to/folder/" (with trailing slash)
  • 'file_list' => array() // Array of sub files and Dirs // Folders Only!

  • 'accessed' => (int) Last accessed timestamp
