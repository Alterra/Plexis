Returns the number of rows affected, or number of rows in the result.

numRows(bool $real = false) : int

This method returns 1 of 2 things. A) either the number of affected rows during the last insert/delete/update query. Or B) The number of rows (count) in the result array.

Parameters

$real   [Default = false]

bool
  • Setting this to TRUE will return The real number of rows. This is not needed unless the last query was a SELECT query, and you are NOT using the mysql driver.

Returns

intReturns the number of rows in the last query
