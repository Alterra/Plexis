    Sets the attribute method for filtering.

    setAttrMethod(int $mode) : void

    Parameters

    $mode

    int

    The filter method. This value will either be the constant value of XssFilter::WHITELIST or XssFilter::BLACKLIST. If set to whitelist, all attributes that are NOT defined will be removed. If set to blacklist, all attributes defined will be removed from the source, and all non-defined attributes will not be filtered out

    Sets the tag method for filtering.

    setTagsMethod(int $mode) : void

    Parameters

    $mode

    int

    The filter method. This value will either be the constant value of XssFilter::WHITELIST or XssFilter::BLACKLIST. If set to whitelist, all tags that are NOT defined will be removed. If set to blacklist, all tags defined will be removed from the source, and all non-defined tags will not be filtered out

    Defines whether the filter should use the blacklist of tags and attributes, and automatically remove them.
